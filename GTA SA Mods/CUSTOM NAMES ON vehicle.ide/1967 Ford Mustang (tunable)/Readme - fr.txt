Ford Mustang Shelby GT500 1967
------------------------------------------------------------------------------------------------
Mod voiture smart - Ford Mustang Shelby GT500 de 1967 pour GTA San Andreas. Le modèle est fait de des textures en haute résolution. Contient les travaux de peinture. La voiture prend également en charge le réglage. 

Caractéristiques: 
- Textures de haute qualité; 
- Détaillée de salon; 
- Contient des travaux de peinture; 
- Support de réglage; 
- L'Adaptation à la FIV; 
- Paramètres propres; 
- Prend en charge toutes les fonctions principales du jeu; 
- La voiture s'intègre parfaitement dans l'ambiance du jeu. 

Vous pouvez télécharger Ford Mustang Shelby GT500 de 1967 pour GTA San Andreas les liens au bas de la page. Installation automatique. Par défaut, ce mod est destiné à remplacer le Remington.

INSTRUCTIONS D'INSTALLATION
------------------------------------------------------------------------------------------------
1. Importer des fichiers dans l'archive IMG
Vous pouvez télécharger IMG Manager 2.0 ici: http://www.gtaall.eu/fr/gta-san-andreas/programs/37425-img-manager-20.html
Crazy IMG Editor ici: http://www.gtaall.eu/fr/gta-san-andreas/programs/3884-gta-sa-crazy-img-editor.html

Utilisez IMG Manager 2.0 ou fou IMG Editor d'importer des fichiers à partir du dossier "01 - Import to gta3.img" pour archiver [JEU DOSSIER]\models\gta3.img:
exh_lr_rem1.dff
exh_lr_rem2.dff
fbmp_lr_rem1.dff
fbmp_lr_rem2.dff
misc_c_lr_rem1.dff
misc_c_lr_rem2.dff
misc_c_lr_rem3.dff
rbmp_lr_rem1.dff
rbmp_lr_rem2.dff
remingtn.dff
remingtn.txd
remingtn1.txd
remingtn2.txd
remingtn3.txd
remingtn4.txd
wg_l_lr_rem1.dff
wg_l_lr_rem2.dff
wg_r_lr_rem1.dff
wg_r_lr_rem2.dff
------------------------------------------------------------------------------------------------
2. Changements dans des fichiers texte

Dans le fichier [Dossier du jeu]\data\vehicles.ide
Trouvez la ligne qui commence par
534, 	remingtn, 	remingtn, 	car, 		REMINGTN, 	REMING, 
Remplacer trouvé ligne avec
534, 	remingtn, 	remingtn, 	car, 		REMINGTN, 	REMING, 	null,	executive,	10, 	0,	0,		-1, 0.75, 0.75,		2

Dans le fichier [Dossier du jeu]\data\handling.cfg
Trouvez la ligne qui commence par
REMINGTN 
Remplacer trouvé ligne avec
REMINGTN 1800.0 4000.0 2.0 0.0 -0.4 -0.2 70 0.75 0.80 0.56 5 160.0 23.0 5.0 R P 6.50 0.50 0 30.0 1.20 0.15 0.0 0.23 -0.04 0.50 0.6 0.21 0.41 30000 40002004 2000000 3 3 1

Dans le fichier [Dossier du jeu]\data\carcols.dat
Trouvez la ligne qui commence par
remingtn, 
Remplacer trouvé ligne avec
remingtn, 3,1, 123,1, 53,1, 0,1, 1,1, 86,1, 2,1, 12,1

Dans le fichier [Dossier du jeu]\data\carmods.dat
Trouvez la ligne qui commence par
remingtn, 
Remplacer trouvé ligne avec
remingtn, exh_lr_rem1, exh_lr_rem2, fbmp_lr_rem1, fbmp_lr_rem2, misc_c_lr_rem1, misc_c_lr_rem2, misc_c_lr_rem3, nto_b_l, nto_b_s, nto_b_tw, rbmp_lr_rem1, rbmp_lr_rem2, wg_l_lr_rem1, wg_l_lr_rem2
