2) Импорт файлов в IMG архив:
С помощью программы IMGTool 2.0 или Crazy IMG Editor импортируйте файлы из папки "To import to gta3.img" в архив [Папка с игрой]\models\gta3.img

IMGTool 2.0 Вы можете скачать тут: https://GameModding.com/ru/gta-san-andreas/gta-sa-programms/829-imgtool-20.html
Crazy IMG Editor тут: https://GameModding.com/ru/gta-san-andreas/gta-sa-programms/828-gta-san-andreas-crazy-img-editor.html
-------------------------------------------------------------
3) Замена настроек для транспорта (Внесите изменения в текстовые файлы):

В файле - [Папка с игрой]\data\handling.cfg
Замените строки с настройками на:
MOONBEAM 1600.0 5848.3 2.5 0.0 0.2 -0.1 85 0.60 0.80 0.50 5 180.0 17.0 15.0 4 D 5.5 0.6 0 30.0 1.4 0.1 0.0 0.35 -0.08 0.55 0.0 0.2 0.75 16000 20 0 1 3 0

В файле - [Папка с игрой]\data\vehicles.ide
Замените строки с настройками на:
418, 	moonbeam, 	moonbeam, 	car, 		MOONBEAM, 	MOONBM, null,	normal,		10, 	0,	0,		-1, 0.80, 0.80,		0

В файле - [Папка с игрой]\data\carmods.dat
Замените строки с настройками на:
moonbeam, exh_b_l,exh_b_m,nto_b_l,nto_b_s,nto_b_tw

В файле - [Папка с игрой]\data\carcols.dat
Замените строки с настройками на:
moonbeam, 119,119, 117,227, 114,114, 108,108, 95,95, 81,81, 61,61, 41,41
