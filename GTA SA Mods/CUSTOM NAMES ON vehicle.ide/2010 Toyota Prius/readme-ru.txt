Изменения в текстовых файлах

В файле [Папка с игрой]\data\vehicles.ide
Найдите строку, которая начинается с
404, 	peren,		peren, 		car, 		PEREN, 		PEREN,	
Замените всю найденную строку на
404, 	peren,		peren, 		car, 		PEREN, 		PEREN,			null,	poorfamily, 	10, 	0,	0,		-1, 0.76, 0.76,		0

В файле [Папка с игрой]\data\handling.cfg
Найдите строку, которая начинается с
PEREN 
Замените всю найденную строку на
PEREN 1370.0 3505.0 2.8 0.0 0.2 -0.1 85 0.60 0.80 0.50 5 180.0 15.0 10.0 F P 5.5 0.6  0  28.0  1.0  0.1  0.0  0.4  -0.13  0.55  0.0  0.2  0.85  20000  20  0   1    1   0

В файле [Папка с игрой]\data\carcols.dat
Найдите строку, которая начинается с
peren, 
Замените всю найденную строку на
peren, 1,1, 3,3, 96,96, 67,67, 103,103, 6,6, 0,0, 99,99

В файле [Папка с игрой]\data\carmods.dat
Найдите строку, которая начинается с
peren, 
Замените всю найденную строку на
peren, nto_b_l, nto_b_s, nto_b_tw
