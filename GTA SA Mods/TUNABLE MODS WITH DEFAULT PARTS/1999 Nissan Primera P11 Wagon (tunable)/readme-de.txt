Nissan Primera Traveller P11
-------------------------------------------------------------
Это моя 100% самостоятельная работа Nissan Primera Traveller P11 - этот универсал я делал примерно 2 месяца, если не считать времени, когда я отдыхал. Всё сделано с любовью и с душой!Модель содержит приблизительно 70000 полигонов, также в модели имеются следующие фичи:- много 3D деталей,- 2 экстры,- работающие стопы и габариты,- адаптированные под High Quality Lights Mod v2,- подсветка приборов,- 4 вида раскраски и возможность тюнинга в TransFender\\\'e,- номерные таблички как в СА,- реальные заводские цвета и многое другое.

2) Importieren von Dateien in das Spiel-Archiv:
Mit dem Programm IMGTool 2.0 oder Crazy IMG Editor importieren Sie Dateien aus einem Ordner "To import to gta3.img" im Archiv [Ordner mit dem Spiel]\models\gta3.img

Die Download-Link fuer IMGTool 2.0 ist hier: https://GameModding.com/en/gta-san-andreas/gta-sa-programms/829-imgtool-20.html
Crazy IMG Editor hier: https://GameModding.com/en/gta-san-andreas/gta-sa-programms/828-gta-san-andreas-crazy-img-editor.html
-------------------------------------------------------------
3) Replacement-Optionen fuer den Transport (Nehmen Sie Aenderungen an Textdateien):

in der Datei - [Installationsordner]\data\handling.cfg
Ersetzen Sie die Zeile mit den Einstellungen mit:
REGINA 1295.0 3000.0 2.0 0.0 0.1 -0.5 75 0.85 0.85 0.48 5 178.0 18.0 15.0 F P 5.0 0.47 1 30.0 0.97 0.1 0.0 0.27 -0.05 0.5 0.2 0.24 0.48 18000 40110020 1400001 0 1 0

in der Datei - [Installationsordner]\data\vehicles.ide
Ersetzen Sie die Zeile mit den Einstellungen mit:
479, 	regina, 	regina, 	car, 		REGINA, 	REGINA, null, richfamily, 10, 0, 0, -1, 0.722, 0.722, 0

in der Datei - [Installationsordner]\data\carmods.dat
Ersetzen Sie die Zeile mit den Einstellungen mit:
regina, nto_b_l,nto_b_s,nto_b_tw,spl_b_bar_m,spl_b_bbr_m,spl_b_mar_m,wg_l_b_ssk

in der Datei - [Installationsordner]\data\carcols.dat
Ersetzen Sie die Zeile mit den Einstellungen mit:
regina, 0,0, 5,5, 17,17, 111,111, 89,89, 90,90

4) Bearbeiten von Textdateien:
Nehmen Sie Aenderungen an Textdateien

in der Datei -  [Installationsordner]\data\carcols.dat
Suchen Sie die Zeile, die
enthaelt: # 0  und
ersetzen Sie sie mit: 16,16,16 # 0 black black

in der Datei -  [Installationsordner]\data\carcols.dat
Suchen Sie die Zeile, die
enthaelt: # 5  und
ersetzen Sie sie mit: 115,82,115 # 5 temple curtain purple purple

in der Datei -  [Installationsordner]\data\carcols.dat
Suchen Sie die Zeile, die
enthaelt: # 17  und
ersetzen Sie sie mit: 181,8,0 # 17 torino red pearl red

in der Datei -  [Installationsordner]\data\carcols.dat
Suchen Sie die Zeile, die
enthaelt: # 111  und
ersetzen Sie sie mit: 173,173,189 # 111 slate gray grey

in der Datei -  [Installationsordner]\data\carcols.dat

enthaelt: # 89  und
