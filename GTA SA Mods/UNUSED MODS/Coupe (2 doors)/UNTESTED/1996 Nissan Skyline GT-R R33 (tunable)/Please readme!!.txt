INSTALLATION

Model replace: Elegy (and tuning parts).

Just replace all the dff and txd files in GTA SA/Models/gta3.img with IMGTool or use Modloader!
You can also replace any other model of the same type of the game (but remember to rename all files correctly)

Done!!
____________________________________________________________________________________________________________________________

For configuration lines (carmods, handling etc), use the default one!!
But you can use these carcols lines, if you want the car to have real colours!

Carcols.dat:
elegy, 122,122, 15,15, 53,53, 43,43, 0,0, 1,1, 3,3, 6,6

These are the original Nissan Skyline colours that I got from Internet!!

Enjoy yourself!
