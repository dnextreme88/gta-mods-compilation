=================================================================
======[How to instal this mod?]====[Kai irasyti sita moda?]======
=================================================================

EN.
=================================================================

1) Extract RAR file.
2) Download IMGTooL or GGMM.
--------------------------------
           IMGTooL
--------------------------------
1) Run IMGTool
2) Open file from GTA SanAndreas Directory "GtaSanAndreas/models/gta3.img"
3) Press "F2" to search
4) Search for file name, one from the RAR
5) Press second mouse buton,and select "REPLACE"
6) Find your Extracted file from RAR,whit same end "TXD or DFF"
7) Hit ok and search for next file,do the same.
8) Then close IMGTool,and have fun whit new model in your GTA SanAndreas.

--------------------------------
            GGMM
--------------------------------
1) Extract GGMM to your GTA SanAndreas Directory
2) Run GGMM
3) Press second mouse button ,and select "Manual Mod Instaler"
4) Select file on the next window
5) Find your Extracted file from RAR
6) After you have selected file "xxxxxx.dff and xxxxx.txd" hit the button INSTAL
7) Close GGMM ,and have fun whit new model in your GTA SanAndreas.

=================================================================

LT.
=================================================================

1) Isekstraktinkite RAR failus.
2) Atsisiuskite IMGTooL arba GGMM.
--------------------------------
           IMGTooL
--------------------------------
1) Paleiskite IMGTool
2) Atidarykite faila is GTA SanAndreas Direktorijos "GtaSanAndreas/models/gta3.img"
3) Spauskite "F2" kad ieskotumete
4) Ieskokite failo pavadinimo, vieno is RAR failu
5) Spauskite antra peles klavisa,ir pasirinkite "REPLACE"
6) Suraskite Isekstraktintus failus is RAR,su tokia pacia galune "TXD arba DFF"
7) Surade spauskite OK,ir ta pati padarykite su kitu failu.
8) Tada uzdarykite IMGTool,ir dziaukites savo nauju modu GTA SanAndreas.

--------------------------------
            GGMM
--------------------------------
1) Isekstraktinkite GGMM i jusu GTA SanAndreas Direktorija
2) Paleiskite GGMM
3) Spauskite antra peles klavisa,ir pasirinkite "Manual Mod Instaler"
4) Pasirinkite failus,isokusiame lange
5) Suraskite isekstarktintus failus is RAR
6) Kai pasirinksite failus "xxxxxx.dff ir xxxxx.txd" paspauskite migtuka INSTAL
7) Uzdarykite GGMM ,ir dziaukites savo nauju modu GTA SanAndreas.
