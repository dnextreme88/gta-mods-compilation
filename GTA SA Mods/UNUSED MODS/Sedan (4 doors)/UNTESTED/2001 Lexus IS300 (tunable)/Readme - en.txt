Lexus IS300 Tuneable
------------------------------------------------------------------------------------------------
Lexus IS300 for GTA San Andreas. Famous sedan from Lexus, which is, I think, was almost all who were on Need For Speed: Most Wanted. Included are wheels and tuning kit of the Most Wanted for greater similarity of the machine from the game! 

Download mod Lexus IS300 for GTA San Andreas below. Mod provides automatic installation and the default is set to replace Savanna.

2. File copying

(!) Do not forget to make copies of the original replaced files to be able to remove the modification!

Copy all the contents of the folder "00 - Copy to game folder" to the folder where the game is installed. Confirm the replacement.

3. Changes in text files

In file [Game folder]\data\vehicles.ide
Find the line that starts with
567, 	savanna, 	savanna, 	car, 		SAVANNA, 	SAVANNA, 
Replace found line with
567, 	savanna, 	savanna, 	car, 		SAVANNA, 	SAVANNA, 	null,	normal,	7, 	0,	2ff0,		-1, 0.65, 0.65,		2

Find the line that starts with
SAVANNA

SAVANNA 1600.0 3300.0 2.2 0.0 0.0 0.0 70 0.70 0.8 0.54 5 235.0 29.0 12.0 R P 5.0 0.60 0 30.0 1.2 0.1 2.0 0.32 -0.19 0.5 0.0 0.0 0.52 19000 40282000 10100000 2 2 0
